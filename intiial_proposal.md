## Goal

Create an application that recommends songs to DJs based on what they are currently playing, using musical attributes such as:

- Tempo  
- Key  
- Danceability  

While usable by anyone, the app is specifically tailored toward DJs.

---

## Minimal Goals

### User Interface
- Landing page  
- Popular songs section  
- Search tab  
- Results page  

### Functionality
- Find songs similar to a provided song (via Spotify API)  
- Display:
  - Tempo  
  - Key  
  - Danceability  
- Provide filtering options:
  - Genre  
  - Decade  
  - Energy  

---

## Stretch Goals

- Song detail card (display detailed song attributes)  
- Under Search tab:
  - Overview of each song  
  - Dedicated view page for each song  
- Include links to play/preview recommended songs  
- History tab (new tab):
  - Display previously searched songs  
- User login  
- Saved playlists  

---

## Project Timeline

### Milestone 1: Core Functionality
- Build UI (landing page, search tab, popular songs)  
- Integrate Spotify API to find similar songs  
- Display tempo, key, danceability  
- Add basic filters (genre, decade, energy)  

### Milestone 2: Enhanced Features
- Add song detail cards and detailed search views  
- Include links to play/preview songs  
- Improve UI/UX navigation and layout  

### Milestone 3: Personalization
- Implement user login  
- Add history tab  
- Enable saved playlists
